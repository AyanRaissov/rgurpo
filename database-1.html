Потребность ушных бирок для сельскохозяйственных животных

База данных по эмиссии индивидуальных номеров взаимосвязана с базой данных по идентификации сельскохозяйственных животных. Ведение базы данных по эмиссии индивидуальных номеров включает: ведение учета выданных индивидуальных номеров сельскохозяйственных животных, распределенных по соответствующим административно-территориальным единицам.
